# 01_web_app/Frontend Agent Guide

## Purpose
Implement the client-facing part of the application using a modern frontend stack.

## Scope
- UI components
- Static assets
- Templates (if server-rendered)
- JS/CSS frameworks (e.g., React, Vite, Tailwind)

## Responsibilities
- Implement and maintain a modern frontend architecture
- Ensure UI usability, responsiveness, and accessibility
- Coordinate with backend APIs
- Keep components modular and maintainable
